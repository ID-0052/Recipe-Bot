# RecipeBot

**RecipeBot** is a web-based AI cooking assistant that generates recipes based on user input. Built with **Streamlit** and **Google Gemini API**, it provides step-by-step instructions and ingredient lists for any dish you ask.

## Features

- Chat-like interface for asking recipes
- Generates concise recipes with ingredients and instructions
- Maintains chat history in sidebar
- “New Chat” button to start fresh conversations

## Technology Stack

- **Frontend & UI**: Streamlit
- **AI Model**: Google Gemini (gemini-2.5-flash)
- **Programming Language**: Python

## Usage

1. Type a dish name or ingredients in the chat box.
2. RecipeBot generates a concise recipe with:
   - Ingredients (with quantities)
   - Step-by-step instructions
3. Sidebar stores previous chats for easy access.
4. Click **New Chat** to start a fresh conversation.

## Note

- Free-tier usage may have limits on API calls and tokens.
